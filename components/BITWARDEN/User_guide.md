# GUIDE D'INSTALLATION 

Déploiement de l'extension Navigateur (Poste Admin XTA-401)
Pour administrer efficacement les comptes, l'extension doit être déployée sur le navigateur Brave/Chrome du poste d'administration :

Ouvre Brave sur le poste d'administration.

Accède au Chrome Web Store et recherche Bitwarden.

Clique sur Ajouter à Brave.

Épingle l'extension dans la barre des tâches pour un accès rapide
