# Installation du package FreeRadius

**Pré-requis :**     

**VM: pfSense**    
**CPU: 1 Core**    
**RAM: 1G**    
**Stockage: 15G**       
**Carte réseau: 3**      

---

Nous utiliserons pfSense pour configurer notre serveur FreeRadius.
Installer pfSense dans une machine virtuelle ici : [pfSense](https://www.pfsense.org/download/)
