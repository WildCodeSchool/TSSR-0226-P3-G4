# PC d'administration : XTA-401

Il est courant d'avoir des machines dédiées à l'administration (systèmes et réseaux).

Les différents outils et logiciels de sécurité listés ci-dessous sont installés, configurés et accessibles (en local ou via le web) depuis ce PC d'administration.

## a. Pour un PC sous Windows

### i. Administration des serveurs Windows

- RSAT => Gestion de consoles serveurs distant (AD, DNS, DHCP, etc.)
- Windows RDP => Prise de main à distance
- Remote PowerShell => CLI à distance
- Serveur RDP => Portail RDP
- Suite logiciels Sysinternal => Suite d'outils

### ii. Administration des serveurs Linux

- GitBash / MobaXterm => Shell bash et SSH/SCP
- Putty / Termius => Connexion SSH
- FileZilla / WinSCP => Transfert de fichiers
- WSL (Windows Subsystem for Linux) => Intégration de Linux dans Windows
- VNC (et dérivés...) => Prise en main à distance GUI

## c. Logiciels et outils multi-OS

- Trippy => Outil de diagnostique réseau 
- OpenSSH => Serveur et client SSH
- Wireshark => Analyse réseau
